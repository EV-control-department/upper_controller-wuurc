# ROV控制系统 V2.0.0 发布说明

## 发布日期

2025年8月1日

## 版本信息

- **版本号**: 2.0.0
- **构建日期**: 2025-08-01
- **打包大小**: 173.87 MB

## 主要功能

1. **模块化设计**：
    - 重构为模块化架构，提高代码可维护性
    - 将功能分离到独立模块，便于扩展和修改

2. **非阻塞操作**：
    - 将阻塞操作改为非阻塞模式，提高系统响应性
    - 使用线程和事件机制优化性能

3. **增强的错误处理**：
    - 增加异常捕获和处理机制
    - 提高系统稳定性和容错能力

4. **优化的视频处理**：
    - 改进视频流处理逻辑
    - 增强帧队列管理

5. **深度温度记录**：
    - 使用非阻塞方式实现深度温度记录
    - 优化数据采集逻辑

## 安装说明

### 系统要求

- **操作系统**: Windows 10/11
- **外部依赖**: FFmpeg（必须安装并添加到系统PATH）
- **硬件要求**: Xbox控制器（推荐使用Xbox One或Xbox Series控制器）

### 安装步骤

1. 解压发布包到任意目录
2. 确保已安装FFmpeg并添加到系统PATH
3. 双击`ROV_Controller.exe`启动应用程序
